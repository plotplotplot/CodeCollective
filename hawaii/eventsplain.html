Claude Code 101

**Claude Code 101**

By the end of this workshop you’ll complete **an A.I. powered web application** using a repeatable Claude Code workflow. You'll also have access to direct support for your project and future projects.

**Examples:**
\- personal portfolio\, freelancer website\, small business website
\- website with user input\, backend logic\, stored data\, and public\-data import
\- web app that collects information from forms\, APIs\, spreadsheets\, or public websites—and organize it into a usable database
\- signup funnels\, contact forms\, lead capture pages\, event registration form
\- searchable directory
\- local resource board

You'll accomplish in a group setting with a starter claude code workspace and hands-on technical help.

*First you publish something, then you make it interactive, then you make it intelligent.*

We’ll use a **requirements-driven** flow: define the job, define “done,” lock constraints—and land a **deployment-ready web app** you can actually use and share.

**Cost:** $100

**What you need:**

* A goal that can be described as a website or web application
* A laptop
* Willingness to follow along and run commands (paint-by-number style)
* Optional: a real workflow/problem you want the app to handle (there will be time reserved to run your own projects!)

**Schedule:**

* **6pm - 7pm:** We’ll get everyone running the software stack locally, and the required accounts set up and connected (anthropic, render, github) and get a static website up
* **7pm - 8pm:** we'll do a few tight iterations of the loop (**requirements → plan → build → run → improve**) until you’ve shipped a deployment-ready baseline of your goal. We’ll repeat the same loop across all three exercises so you leave with a pattern you can reuse on your own projects.
* **8pm - 9pm :** We'll continue the above and optionally apply the same template to **your** workflow/problem. I’ll help you adapt the app(s) to your “X” and get you pointed at a clean next step (and optionally deploy if you’re not already live).
