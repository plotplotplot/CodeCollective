Friday, March 27, 2026

6:00 PM
Claude Code 101

**Claude Code / Codex 101**

By the end of this workshop you’ll complete **three shipped exercises**:

**01 : Static Website**
Build, update, and deploy a live website.
Students learn how to create a clean web presence, maintain it over time, and publish changes confidently.
**Examples:**
\- personal portfolio
\- freelancer website
\- small business website

**02 : Web App**
Turn a website into software with user input, backend logic, stored data, and public-data import.

You’ll learn how to build interactive web apps that collect information from forms, APIs, spreadsheets, or public websites—and organize it into a usable database.

Examples:
\- signup funnel
\- contact form
\- lead capture page
\- event registration form
\- searchable directory
\- local resource board

**03 : A.I. Workflow Web App**
Add LLM-powered workflows to a web app so the software can interpret, generate, and act on user input.
Students learn how to build apps where A.I. helps complete real tasks inside the product, not just chat in a box.
**Examples:**
\- contact form that drafts a response
\- intake form that summarizes requests
\- lead form that qualifies and routes inquiries
\- notes\-to\-email or transcript\-to\-action\-items workflow

*First you publish something, then you make it interactive, then you make it intelligent.*

We’ll use a **requirements-driven** flow: define the job, define “done,” lock constraints—and land a **deployment-ready web app** you can actually use and share.

**Cost:** $100

**What you need:**

* A laptop
* Willingness to follow along and run commands (paint-by-number style)
* Optional: a real workflow/problem you want the app to handle (there will be time reserved to run your own projects!)

**Schedule:**

* **6pm - 7pm:** We’ll get everyone running the software stack locally, and the required accounts set up and connected (anthropic, render, github) and get a static website up
* **7pm - 8pm:** we'll do a few tight iterations of the loop (**requirements → plan → build → run → improve**) until you’ve shipped a deployment-ready baseline. We’ll repeat the same loop across all three exercises so you leave with a pattern you can reuse on your own projects.
* **8pm - 9pm :** We'll continue the above and optionally apply the same template to **your** workflow/problem. I’ll help you adapt the app(s) to your “X” and get you pointed at a clean next step (and optionally deploy if you’re not already live).
