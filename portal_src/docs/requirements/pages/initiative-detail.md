# Initiative detail page requirements

## Purpose

Public, crawlable view of a ballot initiative.

## Must include

- Initiative title
- Initiative status
- Signature count
- Signature goal
- Signature deadline
- Creation date
- Summary
- Expandable section containing first paragraph of initiative text
- Forum section (comments list)
- Topic tags list
- Endorsements list
- Updates list (dated)
- Link/button to reveal related initiatives section
- Link to campaign manager’s public page

## Actions

- Primary: navigate to sign page
- Secondary: save (mock), share (mock), report (mock)

## States

- Not found: show error and path back home.
