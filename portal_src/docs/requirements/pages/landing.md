# Landing page requirements

## Purpose

Provide a clear introduction, primary navigation, and discovery entry point.

## Actors

- Constituent (guest or signed-in)
- Campaign manager

## Must include

- Value proposition and project intent.
- Featured initiatives list (mock data acceptable).
- Search input that filters initiative list (mock).
- Calls-to-action for constituent registration and campaign registration.

## States

- Default: show featured initiatives.
- Search: show filtered initiatives.

## Acceptance criteria

- User can navigate to an initiative detail page.
- User can navigate to constituent/campaign registration pages.
