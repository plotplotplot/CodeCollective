Geocode Cache

Each city publishes its latitude/longitude lookups at /{city}/geocode_cache.json. The shared map page at eventsmap.html reads the cache and event feed for the chosen city using the query parameter ?city=baltimore.

Example links:

  • eventsmap.html?city=baltimore
  • baltimore/geocode_cache.json
  • baltimore/upcoming_events.json

Add new caches by dropping geocode_cache.json files inside each city folder (for example /dc/geocode_cache.json).
